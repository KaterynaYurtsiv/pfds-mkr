# pfds-mkr

Модульна контрольна робота з дисципліни «Python for Data Science».

У роботі виконано:
- підключення до MySQL через Docker-контейнер;
- очищення метеоданих від дублів, NULL та фізичних викидів;
- аналітику даних за допомогою NumPy та Pandas;
- побудову графіків у Matplotlib;
- визначення аномального місяця та аналіз кліматичних показників.

## Структура проєкту

- `solution.py` – основний код роботи
- `plots/` – папка з побудованими графіками

## Використані бібліотеки

- pandas
- numpy
- matplotlib
- sqlalchemy
- mysql-connector-python

## Запуск

```bash
python solution.py
